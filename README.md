# uu

## Installation

```sh
cargo install uu
```

## Usage

```
Usage: uu query maintained [OPTIONS]

Options:
      --by <NAME>
  -n, --not
  -c, --count
  -o, --output <FORMAT>  [default: json] [possible values: json, plain]
  -h, --help             Print help
```
